MDN Web Docs

  • 矢名
  • 2018-5-13
  • 1已阅读

MDN Web Docs（旧称Mozilla Developer Network、Mozilla Developer Center，简称MDN）是一个汇集众多Mozilla产品和网络技术开发文档的免费网站。

MDN Web Docs是 Mozilla 于 2005 年建立的关于 Web API 技术标准的开发文档网站，主要内容包括 HTML, Java, CSS 以及其它项目内容的参考文档和教程，内容由社区开发者和技术文档工程师(Technical Writer)维护。很早以前便成为事实上的开发者 Web 技术参考平台。

参考网址：https://developer.mozilla.org
